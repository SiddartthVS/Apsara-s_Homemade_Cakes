# Gold Diamond Heading Decoration

## Goal
Add gold diamond + gold line decorations flanking section headings (Brownies, Specials, Cakes, Pastries, Live Offers).

## HTML Change
Replace plain `<h2>Title</h2>` with a decorated structure:

```html
<h2 class="decorated-heading">
    <span class="heading-line"></span>
    <span class="heading-diamond">&#9670;</span>
    Title
    <span class="heading-diamond">&#9670;</span>
    <span class="heading-line"></span>
</h2>
```

**File:** `main.html` — update 5 headings:
- Line 211: Brownies
- Line 217: Specials
- Line 223: Cakes
- Line 229: Pastries
- Line 132: Live Offers (in offers section)

## CSS Addition

```css
.decorated-heading {
    display: flex;
```
