# Brand Color Palette Update Plan

## Color Mapping

| Brand Color | Hex | Replaces |
|---|---|---|
| Royal Carpet Red | `#9E1B32` | Badges, strong accents (`#dc3545`) |
| Regal Gold | `#D4AF37` | All gold/orange accents (`#d4a843`, `#d17842`, `#d4a017`, `#b8860b`) |
| Crown Magenta | `#8E254B` | Wave bg accent (`#355c7d`), nav dark base |
| Soft Blush Pink | `#F1C6CD` | Hover/active tints, alternating rows, card shadows |
| Chocolate Brown | `#4E2A1D` | Primary text, nav bg, buttons (replaces `#0c0f14`, `#222`, `#333`, `#111`) |
| Vanilla Cream | `#FCF8F5` | Base backgrounds (replaces `white`, `#f2f2f2`) |

Social platform colors (Instagram, YouTube, WhatsApp) stay as-is.

## File-by-File Changes

### 1. `home/home.css`

**CSS variables** (`.aps-menu-section`):
- `--aps-orange: #d17842` → `#D4AF37` (Regal Gold)
- `--aps-dark: #0c0f14` → `#4E2A1D` (Chocolate Brown)
- `--aps-dark-gray: #52555a` → `#7a5c4f` (Chocolate Brown mid-tone)
- `--aps-gray: #aeaeae` → `#b8a89a` (warm gray matching brown)

**Wave background** (`.wave-bg::before`):
- `--c1: #f8b195` → `#F1C6CD` (Soft Blush Pink)
- `--c2: #355c7d` → `#8E254B` (Crown Magenta)

**Logo** (`.home-logo img`):
- `border: 5px solid #d4a843` → `#D4AF37`
- `box-shadow` gold values → Regal Gold with alpha

**Nav bar** (`.aps-menu-section`):
- `background: rgba(12,15,20,.85)` → `rgba(78,42,29,.88)` (Chocolate Brown)
- `border: 1px solid rgba(255,255,255,.08)` → `rgba(255,255,255,.08)` (keep)
- Active bg `rgba(209,120,66,.12)` → `rgba(212,175,55,.12)` (Regal Gold)

**Sticky nav**:
- `background: rgba(12,15,20,.88)` → `rgba(78,42,29,.90)`
- `border-bottom` → keep or use `rgba(212,175,55,.15)`
