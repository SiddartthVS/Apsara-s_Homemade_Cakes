# Fix: Recent Cakes section not showing images

## Problem
`recent-cakes.js` uses `fetch('.')` expecting a server to return a JSON array of filenames. The site is opened via `file://` protocol with no server, so the fetch fails silently and no images render.

## Fix
Replace the `fetch('.')` approach in `recent-cakes.js` with a hardcoded array of the 8 image filenames that exist in `recent-cakes/`:

```
offer1.jpg, offer2.jpg, offer4.jpg, velvet1.jpg, velvet2.jpg, walnut1.jpg, walnut2.jpg, walnut3.jpg
```

## File to modify
- `recent-cakes/recent-cakes.js` — replace `fetch` call with a static `var images = [...]` array

## Verification
- Open `index.html` in a browser — "Recent Cakes" section should show a scrolling marquee of tilted cake thumbnails
