# Add Tap Hints to Offer Cards

## Changes

### `offers/offers.css` — Add two pseudo-elements

**1. Default state hint on `.card`:**
```css
.card::after {
    content: 'Tap to reveal →';
    position: absolute;
    bottom: 12px;
    left: 50%;
    transform: translateX(-50%);
    font-size: 13px;
    color: rgba(255,255,255,.85);
    text-shadow: 0 1px 4px rgba(0,0,0,.5);
    pointer-events: none;
    z-index: 2;
    transition: opacity 0.3s;
}
```

**2. Hide default hint on hover:**
```css
.card:hover::after {
    opacity: 0;
}
```

**3. Hovered state hint on `.card:hover .card__content`:**
```css
.card:hover .card__content::after {
    content: 'Tap to close <_';
    position: absolute;
    bottom: 12px;
    left: 50%;
    transform: translateX(-50%);
    font-size: 13px;
    color: #4A1520;
    pointer-events: none;
}
```
