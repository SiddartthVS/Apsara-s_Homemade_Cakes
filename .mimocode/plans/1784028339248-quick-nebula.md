# Plan: Add "Recent Cakes" Marquee Section

## Overview
Add a new "Recent Cakes" section between offers and menu — a horizontally scrolling marquee of small tilted cake thumbnails that pauses on hover.

## Files to Modify

| Action | File |
|--------|------|
| Create | `recent-cakes/recent-cakes.css` |
| Modify | `index.html` (add CSS link + section HTML) |
| Modify | `home/home.css` (add `#recent-cakes` to scroll-margin-top) |

## 1. Create `recent-cakes/recent-cakes.css`

```css
#recent-cakes {
    background: #FDF6EC;          /* matches #offers and #menu */
    overflow: hidden;
    scroll-margin-top: 70px;
    padding: 10px 0;
}

/* Smaller heading than the default clamp(2rem, 5vw, 4rem) */
.recent-cakes-heading {
    font-size: clamp(1.4rem, 3vw, 2.4rem) !important;
    margin-bottom: 16px !important;
}

/* Marquee container — clips overflow at viewport edges */
.marquee-wrapper {
    width: 100%;
    overflow: hidden;
    padding: 12px 0;
}

/* Scrolling track — two duplicate sets of images for seamless loop */
.marquee-track {
    display: flex;
    gap: 16px;
    width: max-content;
    animation: marquee-scroll 30s linear infinite;
}

/* Pause on hover (desktop) */
.marquee-wrapper:hover .marquee-track {
    animation-play-state: paused;
}

@keyframes marquee-scroll {
    0%   { transform: translateX(0); }
    100% { transform: translateX(-50%); }
}

/* Individual thumbnail — 1:1 ratio, tilted via --tilt custom property */
.cake-thumb {
    width: 110px;
    height: 110px;
    object-fit: cover;
    border-radius: 12px;
    border: 3px solid #C41E3A;    /* crimson, matches cake-card images */
    flex-shrink: 0;
    cursor: pointer;
    transform: rotate(var(--tilt, 0deg));
    transition: transform .3s ease, box-shadow .3s ease;
}

.cake-thumb:hover {
    transform: rotate(0deg) scale(1.15);
    box-shadow: 0 6px 20px rgba(0,0,0,.25);
    z-index: 2;
}

/* Mobile */
@media (max-width: 768px) {
    .recent-cakes-heading {
        font-size: clamp(1.2rem, 4vw, 1.8rem) !important;
    }
    .cake-thumb {
        width: 85px;
        height: 85px;
        border-width: 2px;
        border-radius: 10px;
    }
    .marquee-track {
        gap: 10px;
        animation-duration: 40s;   /* slower on mobile */
    }
}

@media (max-width: 480px) {
    .cake-thumb {
        width: 70px;
        height: 70px;
    }
    .marquee-track {
        gap: 8px;
    }
}
```

- **`--tilt` values** hand-picked in range -5deg to +5deg for a natural scattered-photos look
